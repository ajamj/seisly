# Plan: Milestone v1.3 - Seismic Visualization

## 1. Context and Objective
The current workstation has a 3D viewport skeleton but lacks the logic to display actual seismic data. This milestone implements the "Section View" and "Map View" (Time Slice) plotting capabilities, bridging the `SeismicVolume` data to the `wgpu` rendering engine.

## 2. Technical Strategy
- **Slicing:** Implement efficient 2D slice extraction from `SeismicVolume` providers.
- **Variable Intensity:** Use a `wgpu::Texture` to hold the slice data and a 1D texture for colormapping in the fragment shader. Store and update this texture in `egui_wgpu::CallbackResources`.
- **Wiggles:** Implement a line-based renderer for wiggle traces, layered over the color section.
- **Responsiveness:** Perform texture uploads in the `egui_wgpu::Callback::prepare` phase to avoid blocking the main UI.
