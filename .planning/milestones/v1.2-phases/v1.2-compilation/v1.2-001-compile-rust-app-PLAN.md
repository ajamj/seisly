# Plan: Compile Rust Workspace

## Context
StrataForge (formerly Seisly) needs to be compiled for Windows using the GNU toolchain. This requires specific PATH configuration and target specification.

## Tasks

### 1. Configure Build Environment
- **Type:** `auto`
- **Description:** Set the $env:PATH to include Scoop shims and GCC.
- **Verification:** `echo $env:PATH`
- **Success Criteria:** PATH contains Scoop and GCC paths.

### 2. Build Workspace
- **Type:** `auto`
- **Description:** Run `cargo build --workspace --target x86_64-pc-windows-gnu`.
- **Verification:** Check exit code of cargo build.
- **Success Criteria:** Build completes successfully.

### 3. Verify Output Binaries
- **Type:** `auto`
- **Description:** Confirm output binaries exist in `target/x86_64-pc-windows-gnu/debug/`.
- **Verification:** `ls target/x86_64-pc-windows-gnu/debug/*.exe`
- **Success Criteria:** Binaries for `sf-cli` and `sf-app` (and others) are found.

## Success Criteria
- [x] All Rust crates compiled without error.
- [x] Binaries generated in the target directory.
