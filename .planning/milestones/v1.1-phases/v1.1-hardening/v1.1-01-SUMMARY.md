# Plan v1.1-01: Shared Memory IPC Foundation - Summary

## Objective

Establish a high-performance Shared Memory (SHM) transport layer for seismic data transfer between the Rust host and Python worker processes.

**Purpose:** JSON-RPC over stdio is the primary bottleneck for large seismic arrays. SHM allows zero-copy or near-zero-copy transfer of multi-gigabyte arrays.
