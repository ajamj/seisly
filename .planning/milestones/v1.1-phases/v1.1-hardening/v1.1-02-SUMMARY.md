**Implementation Details:**

1. **Host-Side Handshake** (`ipc.rs`):
   - Added `PROTOCOL_VERSION` constant set to "1.1"
   - Added `handshake_done` flag to track handshake state
   - Implemented `handshake()` method called automatically before any IPC operation
   - Sends `{ "protocol_version": "1.1", "shm_supported": true }` to worker
   - Validates worker response for version compatibility
   - Logs warning if worker doesn't support SHM
   - Returns clear error on version mismatch

2. **Worker-Side Handler** (`main.rs`):
   - Added `PROTOCOL_VERSION` constant set to "1.1"
   - Implemented `handshake` JSON-RPC method handler
   - Validates host protocol version against supported version
   - Returns error on version mismatch with clear message
   - Returns `{ "protocol_version": "1.1", "shm_supported": true/false }`

3. **New `load_data` Method** (`main.rs`):
   - Handles small data transfers via JSON-RPC
   - Accepts `{ "data": [...], "shape": [...] }` parameters
   - Creates NumPy array and reshapes to target dimensions
   - Returns sum for verification
