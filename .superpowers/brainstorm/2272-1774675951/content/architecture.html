Which "Integration Method" should we start with?

This determines how Rust and Python talk to each other.

A

Embedded Python (PyO3)

Python runs inside the Rust app. Fast data sharing, but more complex setup.

B

Microservice (gRPC/HTTP)

Python runs as a separate process. Easier to debug and update AI independently.
