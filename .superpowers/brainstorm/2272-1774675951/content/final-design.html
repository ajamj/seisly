v0.2 Design Summary: The AI-First Workstation

A robust foundation for autonomous subsurface interpretation.

Key Features

  • 3D Seismic Engine: High-performance wgpu-based volume and slice viewer.
  • AI Microservice: Decoupled PyTorch service for fault detection.
  • Low-Latency Bridge: gRPC for efficient data transfer between Rust and Python.
  • Unified Desktop UI: Professional dark-themed interpretation environment.

Technical Constraints

  • Requires Python 3.10+ and a CUDA-capable GPU for AI acceleration.
  • Initial build will focus on local execution (laptop/workstation).

System Architecture

[Rust Desktop App] ⟷ (gRPC/ProtoBuf) ⟷ [Python AI Service]

  • • Rust handles large memory management (SEG-Y) and real-time rendering.
  • • Python handles 3D CNN inference and model training.
